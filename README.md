# Netflix Clone

A responsive Netflix landing page clone built using HTML and CSS to practice front-end development and improve CSS layout skills.

## Features
- Responsive design
- Hero section
- FAQ accordion (HTML & CSS)
- Footer
- Clean and modern UI inspired by Netflix

## Technologies Used
- HTML5
- CSS3

## Status
✅ Completed

## Purpose
This project was built for learning and practicing:
- Flexbox
- Responsive design
- CSS positioning
- Transitions and hover effects
- Semantic HTML
